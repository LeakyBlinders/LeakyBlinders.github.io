Structured Storage Explorer build 1
This UI is part of the Roland-Structured-Storage-CLI repo, which is a front end for the Roland-Structured-Storage library.

Both of these are currently out of date as they are being refactored, this UI is based on the new version of the library which will be released soon.

For almost everything now instead of lots of sub objects inside things like the an MC-707 project, it is all accessed via the `file.at("...")` path, from the chunk to the actual object.

Not everything is mapped yet, but it is a work in progress.
